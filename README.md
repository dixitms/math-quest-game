# ‚ûï Math Quest: Interactive Learning Game

An interactive, browser-based math game designed for kids to practice operations through gamified mechanics. Built with **Phaser 3**, it features dynamic difficulty, streak multipliers, and local high-score tracking.

## üöÄ Features

### 1. Dynamic Math Engine

The game generates random problems across multiple categories. As the player's score increases, the game **Levels Up**, introducing more complex operations:

* **Level 1:** Basic Addition and Subtraction.
* **Level 2:** Multiplications (up to 12x12).
* **Level 3:** Squares ().
* **Level 4+:** Square Roots () and Cubes ().

### 2. Gamification & Juiciness

* **Streak Multiplier:** Consecutive correct answers increase your point multiplier (up to ). Making a mistake resets the streak.
* **Animations:** Smooth entrance transitions using "Elastic" easing and visual feedback (shakes) for incorrect answers.
* **Sound Synthesis:** Real-time audio generation (Dings, Buzzes, and Level-up chimes) using the Web Audio API‚Äîno external assets required.

### 3. Session Management

* **Dual Timers:** Tracks total session duration and a countdown per question.
* **State Control:** Features to **Pause**, **Restart**, and **Resume** the game at any time.
* **Leaderboard:** Saves high scores and player names to the browser's `localStorage`.

## üì¶ Installation & Local Setup

To run this on your MacBook:

1. **Clone/Download** this repository.
2. Open the folder in **Visual Studio Code**.
3. Install the **Live Server** extension.
4. Right-click `index.html` and select **"Open with Live Server"**.
5. View the game at `http://127.0.0.1:5500`.

---

## üì± Mobile Play

To play on an iPad or iPhone:

1. Navigate to the GitHub Pages URL.
2. Tap the **Share** button in Safari.
3. Select **"Add to Home Screen"** to launch it like a native app without browser UI.

---

## üìù How to Play

1. Click **Start Game**.
2. Solve the math problem displayed at the top.
3. Click the square containing the correct answer before the **Question Timer** hits zero.
4. Build your **Streak** to maximize your score!
